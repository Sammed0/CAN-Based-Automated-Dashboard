# CAN-Based-Automated-Dashboard

 CAN Based Automated Dashboard

## Overview
This project implements an automotive dashboard using CAN (Controller Area Network) communication.

## Features
- Real-time CAN data monitoring
- Vehicle parameter display
- Speed and RPM visualization
- Fault detection
- Embedded C implementation

## Hardware Used
- PIC18F4580
- CAN Transceiver
- LCD Display
- CAN Bus Network

## Software Used
- MPLAB X IDE
- XC8 Compiler
